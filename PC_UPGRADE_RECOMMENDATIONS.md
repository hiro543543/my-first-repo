## 2. 推奨スペック(5年使用前提)

5年間ストレスなく業務に使うため、現時点では「やや上」のスペックを推奨します。

| パーツ | 推奨スペック | 理由 |
| --- | --- | --- |
| CPU | Intel **Core Ultra 7 155H / 165H** 以上、または **Core Ultra 7 Series 2(258V等)**、AMD **Ryzen 7 8845HS / Ryzen AI 7 350** 以上 | 達人・MFクラウド・Excel・Teams等を同時起動しても余裕がある処理性能。Hシリーズ(28〜45W)を選定。 |
| メモリ | **32GB**(LPDDR5x / DDR5) | フリーズ防止の最重要ポイント。16GBは現時点ギリギリ、5年使うなら32GB推奨。 |
| ストレージ | **512GB NVMe SSD(PCIe Gen4)** | 顧客データはMFクラウド+共有サーバー前提で512GBで足りる。読み書きが速いNVMe必須。 |
| ディスプレイ | 14インチ前後 / **WUXGA(1920×1200)以上** / **ノングレア(非光沢)** | 外出時に文字を読みやすく、目が疲れにくい。事務作業はノングレア推奨。 |
| 外部出力 | **Thunderbolt 4 × 2** または USB-C(DP Alt Mode)+ HDMI | モニター2台以上を安定駆動。ドッキングステーション利用なら Thunderbolt 4 必須。 |
| ネットワーク | Wi-Fi 6E / 6 + 有線LAN(あれば) | 事務所LAN接続の安定性確保。 |
| 認証 | Windows Hello(顔認証 or 指紋認証) | 顧客データ保護。パスワード入力の手間削減。 |
| カメラ・マイク | フルHD(1080p)カメラ + ノイズリダクション対応マイク | 週数回のWeb面談の品質向上。 |
| バッテリー | 公称12時間以上 | 外出時の電源確保不要。 |
| 重量 | 1.5kg以下が目安 | 持ち運びはたまにのため、極端な軽量機(1kg以下)は不要。剛性とのバランス重視。 |
| TPM/暗号化 | TPM 2.0 + BitLocker対応 | 顧客データ保護(税理士法上の守秘義務に対応)。 |

## 4. 避けたほうがよい選択

- **メモリ16GB**:現状でも厳しく、5年使用なら確実に不足。32GB一択。
- **HDD搭載モデル**:論外。SSD(NVMe)必須。
- **Snapdragon(ARM)機**:達人のWindows ARM対応が未確認のため避ける。
- **省電力CPU(Uシリーズ・Yシリーズ)のみ**:現在の不満(遅い・フリーズ)を解消できない可能性。Hシリーズ以上推奨。
- **量販店モデルのインチ無印・低解像度(HD/1366×768)**:目への負担大、業務効率も低下。
